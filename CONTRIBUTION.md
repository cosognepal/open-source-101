## Who Can Contribute?

Anyone can contribute:
- First-year bachelor students
- Beginners in HTML
- People new to GitHub
- Anyone interested in open source

You **do not need prior experience** to contribute.

---

## What Can You Contribute?

You can contribute by:
- Adding learning resources
- Improving text or headings
- Adding your name as a contributor
- Improving documentation (README, CONTRIBUTING)
- Adding basic CSS styling

Check the **Issues** tab for tasks labeled:
- `good first issue`
- `beginner`
- `html`
- `documentation`

### Step 2: Clone Your Fork (Optional)

You can either:
- Edit files **directly on GitHub**, OR
- Clone the repository to your computer

